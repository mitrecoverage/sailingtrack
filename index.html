iOS Local Video Player — Telemetry

For very large files, iOS Safari can freeze when playing directly from a blob: URL. This build tries a stronger approach: it uses a Service Worker to serve the selected file via HTTP Range responses at /__localvideo, which often improves buffering on iOS.

Requirements: must be hosted over HTTPS and needs two files: index.html and sw.js. After first install, you may need a one-time reload for the page to be “controlled”.
